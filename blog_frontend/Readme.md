
# Blog Frontend

Frontend for Blog Website


## Route Reference

#### Home page

```http
   /
```

#### Blog Page

```http
  /blog
```


#### Contact Page

```http
  /contact
```

#### About Page

```http
  /about
```
#### Login Page

```http
  /login
```
#### Register Page

```http
  /register
```


## Run Locally

Go to the project directory

```bash
  cd blog_frontend
```

Install dependencies

```bash
  npm install / yarn install
```

Start the server

```bash
  npm run dev / yarn dev
```
